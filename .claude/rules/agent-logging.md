# 에이전트 작업 기록 규칙

**리드 및 팀 에이전트 필수. 서브에이전트(Explore/Plan) 면제.**

## 언제
- git commit 완료 직후
- PR 생성/머지 완료 직후
- 팀 에이전트 해제 직전 (TeamDelete 전)

## 어디: `.claude/memory/sessions/YYYY-MM-DD.md` (append only)

## 날짜 처리
- 00:00~05:00 사이 작업은 직전 날짜 파일에 기록

## 형식
```markdown
### HH:MM - [역할] 작업 제목
- **수행**: 무엇을 했는지
- **파일**: 생성/수정 파일 경로
- **결정**: 설계/기술 결정 (있으면)
- **TODO**: 남은 할 일
```

## 절대 규칙
1. 기록 없이 다음 작업 금지
2. 경로는 프로젝트 루트 기준 상대 경로
3. 기존 내용 삭제/덮어쓰기 금지
4. 시각: `date +%H:%M`
