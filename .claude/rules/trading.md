# 트레이딩 안전 규칙

## 절대 규칙
- API 키/시크릿 소스코드 포함 **금지** (`.env` 또는 DB 암호화 저장)
- 로그에 자격 증명 출력 **금지** (마스킹 필수)
- 기본값 항상 **모의투자** (`is_mock_trading=True`)
- 실거래 전환은 명시적 설정 변경 + 2중 확인

## 주문 안전장치
- 1회 최대 주문 금액 제한
- 일일 최대 주문 횟수 제한 (사용자별)
- 장 운영시간 외 주문 방지
- 가격제한폭(±30%) 검증

## 데이터 처리
- 시세: 타임존 명시 (Asia/Seoul)
- 금액: 정수(원) 단위 (부동소수점 회피)
- 종목코드: 유효성 검증 필수

## 로깅
- 주문 관련 반드시 로깅 (symbol, quantity, price, is_mock)
- 민감 정보 마스킹 (`mask_string()`)
