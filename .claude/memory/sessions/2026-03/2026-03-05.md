#### 4. 백엔드 개발 유용성 — 중요 참고사항 발견
  - **핵심 발견**: 현재 `src/broker/kiwoom.py`와 `src/broker/constants.py`는 실제로 **한국투자증권(KIS) API 형식**을 사용하고 있음

    - 엔드포인트: `/uapi/domestic-stock/v1/...` (KIS) vs `/api/dostk/...` (키움)
