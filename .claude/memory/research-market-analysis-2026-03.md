## 3. 한국 주식시장 단타/스윙 트레이딩 전략 연구

### 3-1. 모멘텀 돌파 전략의 한국 시장 적합성

#### 학술 연구 결과

**한국 시장의 특수한 모멘텀 구조:**
- 한국은 **음(-)의 횡단면 모멘텀**이 관찰되는 시장 (선진국과 반대)
- 외국인 투자자가 활발히 거래하는 종목: 양(+)의 모멘텀 존재
- 외국인 비활성 종목: 과거 수익률과 미래 수익률이 **음(-)의 관련성** → 반전(reversal) 효과
- 2000년 이전: 모멘텀 효과 미관찰, 2000년 이후: 모멘텀 효과 관찰

**변동성 돌파 전략 (래리 윌리엄스):**
- 공식: `당일 매수가 = 시가 + (전일고가 - 전일저가) x k` (k: 0.5~1.0)
- 한국 시장 적용 시 k값 최적화 필요 (한국 시장 특성상 백테스트 필수)
- 한국콘텐츠학회에서 심층신경망 결합 연구 발표

**일중 모멘텀 (Intraday Momentum):**
- KOSPI에서 일중 모멘텀 확인 (30분 단위 수익률과 오버나이트 수익률이 마지막 30분 수익률 예측)
- 다만, 한국은 중국/일본 대비 **약한 증거** (weak evidence)
- 변동성이 거래량보다 일중 모멘텀에 더 강한 영향

**결론: 모멘텀 돌파 전략은 한국 시장에서 조건부로 유효**
- 외국인 거래 활발한 대형주 위주로 적용
- 소형주/저유동성 종목에는 반전 전략이 더 적합할 수 있음
- k값과 진입 조건의 한국 시장 최적화 필수

### 3-2. 평균회귀 전략의 효과

#### 학술 연구 결과

- 한국 시장은 2000년 이후 **평균회귀 패턴** 확인 (variance ratio test)
- 2000년 이전에는 오히려 평균 이탈(mean aversion) 관찰
- **거래량의 역할**: 거래량 변화 효과를 제거하면 평균회귀 특성이 유의하게 감소
- **회귀 속도**: 한국 시장은 주요 시장 중 **가장 느린 평균회귀** → 장기 변동성이 상대적으로 높음
- 단기 수익률 반전(short-term reversal)은 한국 시장에서 일관되게 보고됨

**결론: 평균회귀 전략은 한국 시장에서 유효하나 회귀 속도가 느림**
- 단기(1~5일) 반전 전략에 적합
- 거래량 급증 동반 시 평균회귀 신호가 더 강력
- 느린 회귀 속도를 감안한 보유 기간 설정 필요

- 갭 리스크: 1% 손절 설정 시 41%의 거래가 손절 당하며, 14건은 의도된 가격 너머에서 체결 (평균 -1.95%)

**한국 시장 특성 감안 권장:**
- 단타: 1:1.5~1:2 리스크:보상 비율 (거래세 감안 시 1:2 이상 필요)
- 스윙: 1:2~1:3 리스크:보상 비율
- ATR 기반 동적 손절이 고정 퍼센트 손절보다 효과적

## 4. 쿨다운/리스크 관리 검증

### 4-1. 연속 손실 후 거래 중단의 통계적 유효성

#### 연속 손실 확률 (수학적 필연성)

| 승률 | 100거래 중 3연패 확률 | 4연패 확률 | 5연패 확률 |
|------|---------------------|-----------|-----------|
| 60% | **100%** | 92% | 63% |
| 70% | 93% | 55% | 21% |
| 80% | - | - | 낮음 |

- 승률 60%에서도 100거래 중 5연패 확률이 63% → **연속 손실은 정상적 분산**

#### 연구 결과: 거래 중단이 수익에 미치는 영향

**부정적 증거 (중단이 해로움):**
- Larry Connors 연구: 15~21 거래일 중단 테스트 결과, **4건 중 3건에서 중단 없는 전략이 더 나은 성과**
- Edgeful 분석: 트레이더가 연속 손실 후 전략을 포기하면 "곧 이기기 시작했을 때 바로 그만두는" 경향
- 손절 사용 시 연속 손실 횟수가 3→5회로 증가 (손절이 자연스러운 회복을 차단)

**긍정적 증거 (중단이 유익함):**
- 리벤지 트레이딩 방지: 연속 손실 후 틸트(tilt) 상태 진입 → 충동적 대형 포지션 → 치명적 드로다운
- 리벤지 트레이딩 중단 시: 승률 40-50% → 55-60%, 드로다운 30-40% → 8-12%로 개선
- 일일 손실 한도가 "한 달 전체를 망치는 하루"를 방지

#### 결론: 심리적 편향 vs 통계적 유효성

| 관점 | 판단 |
|------|------|
| **순수 알고리즘** (감정 없음) | 쿨다운 **불필요** — 통계적으로 오히려 수익 감소 |
| **인간 트레이더** (감정 있음) | 쿨다운 **유효** — 틸트/리벤지 트레이딩 방지 효과 |
| **알고 트레이딩 시스템** | 쿨다운보다 **포지션 사이징 조절**이 더 효과적 |

### 4-2. 30분 재진입 금지의 영향

- 직접적인 30분 쿨다운 연구는 부족하나, 관련 증거 종합:
- 쿨다운이 너무 길면 거래 기회 상실 (2시간 이상은 과도)
- 30분은 "정보 소화 시간"과 일치 (학술 연구에서 시장이 새 정보를 소화하는 데 약 30분 소요)
- **권장**: 알고 시스템에서는 시간 기반 쿨다운보다 **조건 기반 재진입** (변동성 감소, 시그널 재확인 등)이 더 합리적

### 4-3. 일일 손실 한도 권장

| 트레이더 유형 | 일일 최대 손실 | 연속 손실 한도 |
|-------------|-------------|-------------|
| 초보 | 0.5~1% | 3회 |
| 경험자 | 평균 수익일 수준 | 3~5회 |
| 알고 시스템 | 2~3% | 포지션 사이즈 축소로 대응 |
| **절대 상한** | **3%** | - |

**Loss-from-top 메커니즘**: 일일 최고 수익 지점에서 손실 한도만큼 하락하면 중단 (수익 보호)

## 5. 한국 시장 거래 시간대별 패턴

### 5-1. U자형 거래량/변동성 패턴

한국 시장도 글로벌 공통의 **U자형 일중 패턴** 확인:

```
거래량/변동성
  │
  │ ▓▓▓▓                              ▓▓▓▓
  │ ▓▓▓▓▓▓                          ▓▓▓▓▓▓
  │ ▓▓▓▓▓▓▓▓                      ▓▓▓▓▓▓▓▓
  │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
  └──────────────────────────────────────────
  09:00   10:00   11:00  12:00  13:00  14:00  15:00
           장시작 30분        점심        마감 30분
```

### 5-2. 시간대별 특성

| 시간대 | 특성 | 전략 적합성 |
|--------|------|-----------|
| **09:00-09:30** | 최고 변동성/거래량. 갭 영향. 정보 소화 기간 | 모멘텀 (단, 노이즈 높음) |
| **09:30-10:30** | 변동성 안정화, 추세 형성 | **모멘텀 돌파 최적** |
| **10:30-13:00** | 거래량/변동성 감소, 점심 시간 | 평균회귀 (범위 거래) |
| **13:00-14:30** | 거래량 감소, 방향성 약함 | 진입 비추천 |
| **14:30-15:30** | 거래량/변동성 재증가, 기관 포지션 조정 | **마감 모멘텀** (일중 모멘텀 연구 확인) |

### 5-3. 일중 모멘텀 연구 결과 (학술)

- **첫 30분 수익률이 마지막 30분 수익률을 예측** (R² = 1.6%)
- 첫 30분 + 12번째 30분(14:30~15:00) 조합 시 R² = 2.6%
- 변동성 높은 날: R² = 3.3%
- 거래 비용 감안 후에도 유의미 (장 마감 근접 시 스프레드 축소)

### 5-4. 현재 시스템 설정 검증

| 현재 설정 | 판단 | 근거 |
|----------|------|------|
| **09:05 이전 진입 차단** | **적절** | 장 시작 5분은 극단적 노이즈. 30분까지가 정보 소화 기간. 다만, ORB 전략은 첫 5분 범위를 기준으로 사용하므로 "관찰은 하되 진입은 09:05 이후"가 합리적 |
| **13:00 이후 신규 진입 차단** | **다소 보수적** | 14:30~15:00에 마감 모멘텀이 있으므로, 13:00 차단 시 이 기회를 놓침. **14:00 또는 14:30 차단이 더 최적**일 수 있음. 단, 현재의 보수적 설정은 낮은 변동성 구간(13:00-14:30) 회피로 거래 비용 절약 효과 |

| 일일 손실 한도 | - | 2-3% 권장 | 2% 설정 |

### 거래 비용 감안 최소 수익률

```
왕복 비용: 0.23%
최소 R:R 1:2, 승률 50% 가정:
  평균 수익 = (0.5 x 익절) - (0.5 x 손절) - 0.23%
  익절 = 2 x 손절 일 때:
  0.5 x 2S - 0.5 x S - 0.23% = 0.5S - 0.23%
  손익분기: S > 0.46% → 손절 최소 0.46%, 익절 최소 0.92%
```

### 위험 요소

4. **외국인 매도**: CME 증거금 개편 + 지정학 리스크로 외국인 순매도 지속
