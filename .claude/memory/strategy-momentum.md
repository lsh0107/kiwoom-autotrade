## 전략 개요
- ~~52주 신고가 근처 종목의 거래량 돌파 시 진입~~ → 이중 전략 (단타/스윙 자동 분류)
- ATR% 기반 변동성 분류: >2% 단타(모멘텀), ≤2% 스윙(평균회귀)
- 진입 필터: 거래량 + 양봉 + 시가상승률 + 진입시간(09:05-13:00) + RSI + **변동성 필터(ATR%<0.35% 스킵)**
- **ATR 동적 SL/TP**: SL=max(ATR%×1.5, 0.5%), TP=max(ATR%×3.0, 1.0%) — R:R 1:2 ✅ 구현
- 트레일링 스톱 ✅ 구현
- **리스크 관리**: 2연패→50% 축소, 3연패→블랙리스트, kill_switch(-2%/-3%), 섹터당 1포지션 ✅ 구현
- 타임프레임: 5분봉 기반 당일 청산 15:15 (단타) / 5거래일 이내 (스윙, Phase 2)
- LLM: 장전 브리핑(의사결정 보조), 장후 리뷰 (Phase 3)

## 진입 조건 (v2.0)
1. ~~52주 신고가의 `high_52w_threshold`% 이상~~ → 비활성 (단타에 부적합)
2. 당일 거래량 >= 최근 20일 평균 거래량 × `volume_ratio` × `time_ratio`
3. 양봉 필터: 현재 봉 close > open (bar_open 필요)
4. 시가 상승률: 현재가 > 당일 시가 × (1 + price_change_min) (day_open 필요)
5. 진입 시간 필터: 09:05 ~ 13:00 (current_time 필요)
6. RSI 필터: RSI >= rsi_min (옵션)
7. **변동성 필터**: ATR% < 0.35% → 진입 스킵 (거래비용 대비 변동성 부족)
