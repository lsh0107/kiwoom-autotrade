# 세션 메모리 저장

현재 세션에서 수행한 작업을 정리하고 메모리에 저장하세요.

## 수행 단계

1. **세션 로그 파일 확인**: `.claude/memory/sessions/$(date +%Y-%m-%d).md` 파일을 읽거나 새로 생성
2. **현재 세션 요약 작성**: 아래 형식으로 append

```markdown
### HH:MM - [작업 제목]
- **수행**: 이번 작업에서 수행한 내용 요약
- **변경 파일**: 변경/생성/삭제된 파일 목록
- **결정**: 내린 주요 결정사항
- **이슈**: 발견된 문제나 주의사항
- **다음**: 다음에 할 작업
```

3. **프로젝트 상태 업데이트** (필요시): `.claude/memory/project.md` 업데이트
4. **아키텍처 결정 기록** (필요시): `.claude/memory/architecture.md` 업데이트

## 규칙
- 세션 로그는 해당 날짜 파일에 시간순으로 append
- 이미 존재하는 내용은 절대 삭제하지 않고 아래에 추가
- 간결하되 나중에 다시 봤을 때 맥락을 이해할 수 있도록 작성
- 한글로 작성
