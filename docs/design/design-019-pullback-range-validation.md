## 6. 누적 폐기 4건 패턴 분석

2026-04-27 현재까지 KOSPI 대형주 유니버스에서 네 번의 전략/파라미터 검증이 전부 실패했다.

| # | 전략 | 폐기 사유 | PR |
|---|------|----------|-----|
| 1 | 5분봉 모멘텀 | 왕복비용 0.53% × 일 5거래 → 일 -2.65% 비용 구조 | #325~#332 |
| 2 | 52주 신고가 일봉 (기본 grid) | 20종목 0/20 — OOS/IS 일관성 붕괴(19/20), RR 구조 한계(15/20) | design-016 |
| 3 | 52주 신고가 + atr_tp_mult/tp_pct 재검증 | 400 시나리오 0/20 — 파라미터 탐색 공간 소진 | #333 (ADR-018) |
| 4 | Pullback/Range/MR 일봉 어댑터 | 12 combo × 20종목 0/20 — 신호 희소 + RR 미달 | #337 (ADR-019) |

### 공통 패턴 (구조적 가설)

1. **KOSPI 대형주 유니버스의 신호 희소성**: 시가총액 상위 20종은 하루 변동폭이 좁아 Pullback/Range 신호 조건이 충족되는 일이 드물다. OOS 기간에 거래가 수 건에 불과하면 Sharpe는 필연적으로 0에 수렴.

2. **OOS/IS 일관성 붕괴**: IS 기간 학습된 파라미터가 OOS에서 재현되지 않음. KOSPI 레짐이 IS ↔ OOS 간 빠르게 전환되어 어떤 단일 파라미터도 양 기간을 동시에 포괄하지 못한다.

3. **보수적 판정 기준 vs 현 유니버스**: Sharpe ≥ 1.0 + RR ≥ 2.0 기준은 S&P500/나스닥 수준의 변동성이 있는 자산에서 설계됐다. KOSPI 대형주는 일간 ATR이 작아 RR 2.0 달성 자체가 구조적으로 어렵다.

4. **거래비용 임계**: 커미션 0.015% + 세금 0.2% 왕복 ≈ 0.43%. 승률 35%, RR 2.0 기준 손익분기 승률은 33%(RR 2.0 기준 1/(1+2)). 시장 이동이 작으면 익절이 닫히기 전 비용이 수익을 잠식한다.

## 8. 후속 옵션 비교

ADR-018 옵션 B(Pullback/Range walk-forward)가 실패한 상황에서 네 가지 방향이 남는다.

### 옵션 A: KOSPI 소형/중형주 유니버스 재검증

| 항목 | 내용 |
|------|------|
| 전제 | 대형주 신호 희소 → 소형주는 ATR 폭 크고 Pullback/Range 조건 충족 빈도 높음 |
| 장점 | GenericDailyEngine 재활용 가능. 전략 코드 변경 없음. 유니버스만 교체 |
| 단점 | 소형주 유동성 리스크 ↑ (슬리피지 실제로 더 클 것). 체결 불확실성 ↑ |
| 선행 조건 | 소형주 20종목 선정 기준 정의 (시총 1조 미만 + 일 거래대금 50억 이상) |
| 소요 | 중간 (데이터 수집 + walk-forward 재실행, 엔진 불변) |
| 리스크 | 소형주도 KOSPI 레짐 영향 동일 → OOS/IS 붕괴 패턴 반복 가능 |

### 옵션 C: 레짐 필터 강화 (BULL_STRONG에서만 진입)

| 항목 | 내용 |
|------|------|
| 전제 | TREND_BULL_STRONG 레짐에서 Pullback 신호 품질이 가장 높다는 가설 |
| 장점 | 기존 전략 + design-013 multi-regime 아키텍처 활용. 코드 최소 변경 |
| 단점 | 레짐 필터로 신호 더 줄어듦 → OOS 거래 수 감소 → 통계 형성 더 어려움 |
| 선행 조건 | BULL_STRONG 기간만 추출한 walk-forward 결과 필요 |
| 소요 | 낮음~중간 (레짐 마스킹 추가 후 재실행) |
| 리스크 | 신호 수 감소로 통계적 유의성 더욱 약화 가능 |

### 옵션 D: 판정 기준 완화 (Sharpe ≥ 0.5, OOS/IS ≥ 0.5)

| 항목 | 내용 |
|------|------|
| 전제 | 현 기준(Sharpe 1.0, OOS/IS 0.7)이 KOSPI 대형주에 지나치게 엄격하다는 가설 |
| 장점 | 추가 개발 없이 기존 결과 재평가 가능. 즉시 실행 |
| 단점 | **거짓 통과 위험**: 기준을 낮추면 수익성이 없는 전략이 통과될 수 있음. 모의투자에서 실제 손실 발생 가능 |
| 선행 조건 | 완화된 기준으로 통과한 전략의 실 수익성 사전 검토 필요 |
| 소요 | 매우 낮음 (기준값만 변경) |
| **핵심 리스크** | **현재 Sharpe ≈ 0.0, RR < 0.6 — 기준을 0.5로 낮춰도 통과 종목 없음. 실효성 없음** |

### 옵션 E: 다른 패러다임 (페어 트레이딩, ETF 회전, 옵션)

| 항목 | 내용 |
|------|------|
| 전제 | 현 단일 종목 일봉 패러다임이 KOSPI 대형주에 구조적으로 부적합 |
| 장점 | 근본적으로 다른 알파 소스 탐색. OOS/IS 일관성 문제 회피 가능 |
| 단점 | 대규모 재설계. 기존 엔진/전략 코드 대부분 교체. 학습 비용 높음 |
| 선행 조건 | 전략 방향 결정 → ADR 작성 → 엔진 재설계 → 검증 사이클 전체 재시작 |
| 소요 | 높음 (수 주 이상) |
| 리스크 | 불확실성 가장 높음. 단, 구조적 전환이 필요하다면 가장 확실한 방향 |

### 권고 순서

**1순위 옵션 A → 2순위 옵션 C → 필요시 옵션 E**

| 순위 | 옵션 | 이유 |
|------|------|------|
| 1 | A (소형주 유니버스) | 전략·엔진 재활용 가능. 신호 희소성 문제를 유니버스 교체로 직접 해결 |
| 2 | C (레짐 필터) | 추가 데이터 불필요. 다만 신호 감소로 A 실패 시 보조 검증 역할 |
| 보류 | D (기준 완화) | 현재 Sharpe/RR 수준에서 완화해도 통과 불가 — 실효성 없음 |
| 장기 | E (패러다임 전환) | A/C 모두 실패 시 검토. 현 시점에서는 너무 큰 비용 |
