# Design 016: 전략 재설계 — 5분봉 폐기 + 52주 신고가 일봉 채택

## 1. 결정

**5분봉 단타 모멘텀 폐기 → 52주 신고가 일봉 모멘텀 채택**

단, T5 20종목 walk-forward 결과 **전체 기준 통과 0/20 (0%)** → **모의투자 재개 전 파라미터 재조정 필수**

## 2. 5분봉 폐기 근거

### 2-1. 왕복 비용 분석 (T1 엔진 보정 후)

| 항목 | 수치 |
|------|------|
| 슬리피지 (왕복) | 0.30% |
| 수수료 (왕복) | 0.03% |
| 거래세 (매도) | 0.20% |
| **총 왕복 비용** | **0.53%** |
| 일평균 거래 건수 | 5건 |
| **일간 비용** | **2.65%** |
| **월간 비용** | **~58.3%** |

결론: T1 보정 슬리피지 0.15% 적용 시 5분봉 전략은 하루 5건 거래만으로 월 58% 비용 발생 → **breakeven 불가**.

### 2-2. 백테스트 성과 (T1 엔진, 005930 삼성전자 18개월)

| 지표 | T1 이전 | T1 이후 |

| Sharpe | 미측정 | -15.86 |
| MDD | -53.3% | -53.3% |
| 수익인자 | 0.07 | 0.07 |

### 2-3. 이론적 한계

- 한국 증시 일중 패턴: 09:00~09:30 고변동, 점심 저유동, 14:00 이후 변동 재개
- 5분봉 고빈도 매매는 HFT 레이어와 경쟁 → 개인 투자자 불리
- 강장구·이우백 (2016) — KOSPI 52주 신고가 모멘텀 실증 연구 근거

## 3. 52주 신고가 일봉 전략

### 3-1. 진입 신호

```
1. 당일 종가 > 직전 lookback일 최고가 (look-ahead 방지: prior_daily 사용)
2. 당일 거래량 > 20일 평균 × vol_mult
3. KOSPI 종가 > KOSPI 20MA (시장 방향 필터)
```

### 3-2. 청산 조건 (우선순위 순)

1. ATR 1.5x 손절 (`stop_loss`)
2. ATR 4x 또는 +5% 익절 (`take_profit`)
3. Trailing stop (+2% armed → 고점 대비 -1% 하락, `trailing_stop`)
4. 최대 보유 10거래일 (`max_holding`)

### 3-3. 기본 파라미터

| 파라미터 | 기본값 | 그리드 |
|----------|--------|--------|
| lookback | 20 | 10/20/30 |
| vol_mult | 1.5 | 1.2/1.5/2.0 |
| atr_stop_mult | 1.5 | 1.0/1.5/2.0 |
| atr_tp_mult | 4.0 | 3/4/6 |
| slippage_pct | 0.0015 | 고정 |

### 3-4. 대안 전략 비교

|------|------|------|------|
| 5분봉 단타 모멘텀 | ~17% | 2.65%/일 | **폐기** |

### 4-4. 핵심 진단

**주요 실패 원인: RR(Risk-Reward) 0% 통과**

현재 파라미터(ATR 1.5x 손절, 4x 익절)에서 KOSPI 대형주 특성상 avg_win/avg_loss 비율이 최대 1.3 수준:
- ATR 기반 익절이 +5% 고정 상한에 막혀 큰 수익 기회 절단
- 손절은 ATR 변동성에 맞게 작동하나 익절 상한이 RR을 제한
- 대형주(시총 10조+)는 단기 모멘텀 폭이 소형주 대비 제한적

**보조 실패 원인: MDD 85% 실패**

- 다수 종목에서 MDD -12~-29% 기록
- 현재 리스크 가드레일(T3: HWM -5% STOP_BUY, -7% FORCE_CLOSE)은 포트폴리오 수준 보호
- 개별 종목 MDD는 위 가드레일이 직접 제어하지 않음

**긍정적 신호**:
- 승률 100% 통과 (전 종목 35%+, 평균 59%)
- OOS Sharpe 50% 통과 — 10종목에서 실제 OOS 성과 양호
- LG에너지솔루션(6.36), KB금융(4.50), 셀트리온(3.42) 등 OOS Sharpe 우수

## 5. 권고사항

### 즉시 조치 (필수)

1. **파라미터 재조정 — atr_tp_mult 상향** (`4.0 → 6.0 또는 tp_pct 제거`)
   - 목적: RR 개선. 익절 상한 +5% 제거 또는 상향 → avg_win 증가
   - 예상 효과: RR 1.3 → 2.0+ (이론적)

2. **손절 완화 — atr_stop_mult 상향** (`1.5 → 2.0`)
   - 목적: MDD 개선은 아니지만 조기 손절로 인한 재진입 비용 최소화

3. **MDD 기준 완화 고려** (`-10% → -15%`)
   - 근거: KOSPI 대형주 자체 변동성이 -15% 이상 구간 존재. 종목 MDD와 포트 MDD는 별개
   - 전략 폐기 기준을 포트폴리오 MDD로 재정의하는 것이 더 현실적

### 중기 조치

4. **종목 선별 — 유망 10종목으로 축소**
   - OOS Sharpe ≥ 1.0 + MDD ≤ -15%: LG에너지솔루션, 셀트리온, 기아, SK하이닉스, 삼성SDI, KB금융, 하나금융지주, SK, 포스코퓨처엠, LG화학

5. **파라미터 재조정 후 재검증**
   - 수정 파라미터로 walk-forward 재실행, 통과 30%+ 달성 확인 후 모의투자 재개
